Prepare social media posts (Facebook & LinkedIn) for a visual guide.

1. **Verify Inputs**: Check which guide the user wants to post about. If not specified, ask. Default language: English.

2. **Analyze the Guide**: Read the guide component, extract core concepts and takeaways, determine the URL.

3. **Draft Posts**:
   - **Tone**: Natural, conversational, informal — like sharing a personal experience.
   - **Humble language**: Use "might help", "could be useful" instead of absolutes.
   - **Structure**: Hook (relatable problem/question) → Context (why it matters) → Solution (the guide) → CTA (low-pressure invite).
   - **LinkedIn**: Story-telling but slightly professional. 3-5 hashtags including `#Vizly`.
   - **Facebook**: Highly conversational with relevant emojis. 3-5 hashtags including `#Vizly`.

4. **Multi-Language**: If requested (e.g., Bangla), translate capturing the informal story-telling essence — not a robotic translation.

5. **Present**: Output with clear Markdown headings for easy copy-paste.
